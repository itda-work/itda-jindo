## 현재 문제점

1. **설치가 되지 않음**: Space로 선택 후 Enter를 눌러도 ~/.claude/skills/에 파일이 생성되지 않음
2. **피드백 부재**: Enter 후 화면이 그대로 유지되어 설치 진행/완료 여부를 알 수 없음
3. **체크 아이콘 혼동**: 녹색 ✓가 '이미 설치됨'인지 '선택됨'인지 불명확

## 기대 동작

1. 설치 진행 시 모달/팝업으로 진행 상황 표시
2. 녹색 ✓는 '이미 설치됨' 상태를 의미
3. 선택됨은 별도 표시 방식 필요 (예: [x] vs [✓] 구분)
4. 실제 설치가 정상적으로 이루어져야 함
